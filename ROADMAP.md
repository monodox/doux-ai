- Stabilize local CLI and Gradio UX
- Harden Ollama compatibility across local configurations

- Collaborative local sessions and artifact versioning
- Rich run observability and replay
- Deeper TypeScript SDK parity with Python APIs
- Optional distributed execution while keeping local-first defaults
